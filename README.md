# microservice-fiap-process-video

## Pré-Requisitos
Este projeto foi feito com:
- .NET 8
- Visual Studio 2022

## Variáveis de Ambiente
Criar um .env dentro da pasta .\FiapProcessaVideo.WebApi e preencher as variáveis de ambiente a seguir:
PORT=
S3_BUCKET_NAME=
AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
AWS_ACCESS_TOKEN=
AWS_REGION=
